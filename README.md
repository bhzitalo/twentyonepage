# Twenty One Pilots — Página Web

Este projeto é uma página web desenvolvida como atividade do curso de Desenvolvimento de Sistemas, utilizando HTML, CSS e JavaScript.

A página apresenta informações sobre a banda Twenty One Pilots, come elementos visuais como imagem de capa, carrossel de imagens e uma tabela informativa.

O objetivo do projeto é praticar HTML, a estilização com CSS e a aplicação de interações simples com JavaScript.
